## Projects

### `neomovies-web`
Web client for NeoMovies.
- Repo: https://github.com/Neo-Open-Source/neomovies-web
- Stack: React, TypeScript, Vite
- Purpose: Fast web UI for browsing and watching

### `neomovies-api`
Backend API and docs for NeoMovies.
- Repo: https://github.com/Neo-Open-Source/neomovies-api
- Stack: Go, MongoDB, VitePress docs
- Purpose: Search, metadata, players, auth and integrations

### `neomovies-android`
Android client for NeoMovies.
- Repo: https://github.com/Neo-Open-Source/neomovies-android
- Stack: Kotlin, ExoPlayer/libMPV
- Purpose: Native mobile app with streaming and TorrServer support
